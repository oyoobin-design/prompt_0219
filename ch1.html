프롬프트 유형 & 샷 기법
프롬프트를 어떤 방식으로 구성하느냐에 따라 AI의 응답 품질과 재사용성이 달라진다. 3가지 유형과 예시 수에 따른 샷 기법을 익히면 목적에 맞는 프롬프트를 골라 쓸 수 있다.
프롬프트의 3가지 유형
유형재사용율유연성특징
서술형낮음높음토큰 많이 사용 / 맥락 전달에 강점
지침형중간중간규칙 기반 출력 제어에 적합
함수형높음낮음토큰 절약 / 반복 자동화에 강점
서술형 Descriptive
맥락과 배경을 문장으로 서술하여 AI가 상황을 이해하도록 유도
너는 10년차 무역 전문가야. 중소기업 수출 담당자가 HS code를 처음 배우는 상황이야. 이 사람에게 HS code의 개념과 중요성을 설명해줘.
AI 예상 응답
HS Code란 '국제통일상품분류체계'로, 전 세계 200여 개국이 공통으로 사용하는 상품 분류 코드입니다. 수출할 때 이 코드를 정확히 써야 관세율이 결정되고, 통관도 가능합니다. 틀리면 통관이 거부되거나 추가 세금이 부과될 수 있으니 첫 수출 전 반드시 확인하세요...
지침형 Instructive
단계별 절차나 명확한 규칙을 번호 목록으로 제시하여 원하는 출력을 유도
다음 규칙에 따라 python 코드를 작성해. 1. 함수명은 snake_case로 작성 2. 각 함수에 docstring 포함 3. 예외 처리 반드시 포함 4. 결과는 json 파일로 만들기
AI 예상 응답
def calculate_tax(price: float) -> dict: """세금을 계산하여 JSON으로 반환합니다.""" try: result = {"tax": price * 0.1, "total": price * 1.1} return result except TypeError as e: raise ValueError(f"유효하지 않은 입력값: {e}")
함수형 Functional
입력값(변수)과 출력값(구조)을 명확히 정의하여 일관된 결과를 반복 생성
입력 : {반도체}, {8501}, {대만} 처리 : 해당 국가의 관세율 및 수입 규제 조회 출력 : { "관세율" : " ", "규제사항" : [ ], "필요서류" : [ ] }
AI 예상 응답
{ "관세율" : "0% (MFN 세율 적용 / 한-대만 FTA 미체결)", "규제사항" : ["전략물자 해당 여부 확인 필요", "수출허가서 필요"], "필요서류" : ["상업송장", "포장명세서", "원산지증명서", "수출허가서"] }
샷(Shot) 프롬프팅 기법
AI에게 제공하는 예시 수에 따라 패턴 학습 정확도가 달라진다. 예시가 많을수록 원하는 형식을 더 정확히 모방한다.
제로샷 Zero-shot
예시 없이 바로 지시 — 단순하거나 AI가 이미 충분히 학습한 유형에 적합
프롬프트
원샷 One-shot
예시 1개 제공 — 출력 형식이 단순하거나 명확한 경우에 적합
감정을 분류해줘. 예시) "배송이 너무 늦어 실망했어요" → 감정 : 부정 처리: "이 제품 정말 마음에 들어요!"
AI 예상 응답
감정 : 긍정
퓨샷 Few-shot
예시 2개 이상 제공 — 형식이 복잡하거나 일관성이 중요한 도메인 특화 작업에 적합
HS code를 분류해줘. [예시] 품목 : "면으로 만든 남성용 티셔츠" → 6109.10 품목 : "폴리에스테르 소재 여성 블라우스" → 6206.40 품목 : "아동용 데님 바지" → 6203.42 [처리] 품목 : "{분류할 품목명}" →

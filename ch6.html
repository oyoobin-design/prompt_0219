레시피 · 뒤집힌 상호작용 · 인지 검증자
실행 가능한 단계별 지침을 받는 레시피 패턴, AI가 먼저 질문하는 뒤집힌 상호작용 패턴, 복잡한 문제를 하위 질문으로 분해하는 인지 검증자 패턴.
레시피 패턴
일반 질문은 두루뭉실한 방향만 제시하지만, 레시피 패턴은 목표·현재 상태·출력 형식을 명시하여 즉시 실행 가능한 순서별 행동 지침을 생성한다. 혼자 진행할 때 막막함을 없애주는 핵심 패턴.
방식예시 질문결과
일반 질문"수출 자동화 어떻게 해?"개념 설명 + 두루뭉실한 방향 (실행 불가)
레시피 패턴"수출 자동화를 위한 단계별 절차를 주세요"순서별 행동 지침 (즉시 실행 가능)
유형 1: 기본 단계형 STEP 형식
# 목표 python 초보자가 무역 HS code 자동 분류 프로그램 만들기 # 현재 상태 - python 설치만 되어 있음 - 프로그램 개발 경험 없음 - 엑셀로 품목 데이터 500개 보유 # 요청 위의 목표를 달성하기 위한 단계별 레시피를 작성하세요. ## STEP {번호} : {단계명} - 할 일 : {구체적 행동} - 소요시간 : {예상 기간} - 완료 기준 : {다음 단계로 넘어갈 기준} - 주의사항 : {흔한 실수}
AI 예상 응답 구조
## STEP 1 : 환경 세팅 - 할 일 : pip install pandas openpyxl 실행 - 소요시간 : 30분 - 완료 기준 : import pandas as pd 실행 후 오류 없음 - 주의사항 : 파이썬 PATH 설정 안 하면 pip 명령이 안 됨 ## STEP 2 : 엑셀 데이터 불러오기 - 할 일 : pd.read_excel('품목데이터.xlsx') 코드 작성 - 소요시간 : 1시간...
유형 2: 분기 포함형 정상/이상 분기
신규 직원 온보딩 프로세스를 단계별로 설명해줘. 단, 각 단계에서 발생할 수 있는 분기 상황도 포함하세요. 형식: STEP N. {행동} - 정상 : {이 결과가 나오면 → 다음 단계로 진행} - 이상 : {이 결과가 나오면 → 어떻게 대응해야 하는지}
AI 예상 응답 구조
STEP 1. 입사 서류 제출 및 시스템 계정 생성 - 정상 : 서류 전부 수령 완료 → STEP 2로 진행 - 이상 : 서류 미비 시 → HR 담당자 연락, 2일 이내 재제출 요청 STEP 2. 부서 배치 및 멘토 연결 - 정상 : 멘토 지정 완료, 첫 미팅 일정 확정 → STEP 3으로 진행 - 이상 : 멘토 부재 시 → 팀장이 임시 담당, 1주 내 재배정 STEP 3. OJT 교육 진행 (1~2주) - 정상 : 매뉴얼 숙지 확인, 실습 완료 → 정식 업무 투입 - 이상 : 이해도 부족 시 → 교육 1주 연장, 멘토 집중 코칭
유형 3: 체크리스트형 누락 방지
유튜브 쇼츠 영상 업로드를 위한 실행 체크리스트를 작성해줘. 형식: [ ] {실행해야 할 행동} - {왜 해야 하는지 또는 주의사항}
AI 예상 응답 구조
[ ] 영상 길이 60초 이내로 편집 - 초과 시 쇼츠 피드에서 제외됨 [ ] 세로 비율(9:16) 확인 - 가로 영상은 쇼츠 최적화 불가 [ ] 첫 3초 내 훅(Hook) 삽입 - 이탈율을 낮추는 핵심 요소 [ ] 자막 삽입 여부 확인 - 무음 시청자 대응, 시청 유지율 증가 [ ] 썸네일 직접 업로드 - 자동 선택 대비 클릭률 2~3배 향상 [ ] 제목에 핵심 키워드 포함 - 검색 노출 및 추천 알고리즘 영향 [ ] 해시태그 3~5개 추가 - #Shorts 필수 포함, 과다 태그 역효과
유형 4: 병렬 레시피형 타임라인
여러 작업을 동시에 진행할 때 — "지금 뭘 해야 하지?" 고민하는 시간을 없애준다
신규 앱 출시 준비를 달성하기 위한 2주간의 병렬 실행 계획을 작성하세요. 각 날짜마다 동시에 진행 가능한 작업들을 병렬로 배치하세요. 형식: | 1일차 | 2일차 | 3일차 | |-----------|-----------|-----------| | ㅁ 작업 A | ㅁ 작업 C | ㅁ 작업 E | | ㅁ 작업 B | ㅁ 작업 D | ㅁ 작업 F |

뒤집힌 상호작용 패턴
일반: 사용자가 정보를 주면 AI가 즉시 결과 생성 → 정보 부족으로 부정확한 결과물 생성 위험
이 패턴: AI가 먼저 필요한 정보를 질문 → 정보가 충분히 수집된 후에만 결과물 생성
방법 1: 일괄 질문형 수집 항목 명확할 때
제가 원하는 결과물은 수출 제안서입니다. 바로 만들지 말고, 먼저 아래 항목들을 질문해서 제 답변을 모두 수집한 후 작성하세요. 수집이 필요한 정보: - 제품 정보 (품목명, 스펙, 주요 특징) - 타깃 바이어 (국가, 업종, 규모) - 핵심 강점 (경쟁사 대비 우위) - 가격 정책 (희망 단가, 최소 주문량) - 납기 조건 (생산 리드타임, 선적 조건)
방법 2: 순차 질문형 대화형 정보 수집
제가 원하는 결과물은 python 학습 계획서입니다. 한 번에 하나의 질문만 하세요. 제가 답하면 다음 질문으로 넘어가고, 충분한 정보가 모이면 결과물을 작성하세요. 질문은 5개의 선택지를 제시하고 번호로 선택 가능하게 해주세요. 5~7개의 질문이 끝나면 맞춤형 학습 계획서를 작성해주세요.
항목방법 1 (일괄 질문)방법 2 (순차 질문)
질문 방식한 번에 모든 질문 제시한 번에 하나씩
적합한 상황수집 항목이 미리 정해진 경우항목이 유동적이거나 대화형
사용자 부담여러 항목 한꺼번에 답해야 함하나씩 답하면 되어 부담 적음
인지 검증자 패턴
복잡한 질문을 바로 입력하면 AI가 즉각 답변하면서 오류·누락 가능성이 높아진다. 이 패턴은 AI가 먼저 문제를 하위 질문으로 분해하고 각 질문을 순서대로 검증한 뒤 통합된 판단을 내리도록 유도한다.
핵심 문구: "바로 답하지 말고" — 이 한 문장이 AI의 즉각 응답을 억제하고 단계적 검증 모드로 전환시킨다.
방법 1: 명시적 분해 요청
아래 질문에 바로 답하지 말고 먼저 이 질문을 정확히 답하기 위해 필요한 하위 질문을 3~5개 도출하세요. 각 하위 질문에 순서대로 답한 뒤 최종 통합 답변을 제시하세요. 질문: 우리 회사가 베트남에 생산 법인을 설립하면 세금, 인력, 물류 측면에서 유리한가?
AI 예상 응답 구조
하위 질문 도출: Q1. 베트남의 법인세율과 한국 대비 절세 효과는? Q2. 베트남 현지 인력 수급 상황과 인건비 수준은? Q3. 한국→베트남 물류 비용 및 리드타임은? [Q1 검증] 베트남 법인세율 20%, 한국 대비... [Q2 검증] 평균 인건비 월 350달러 수준으로... [Q3 검증] 부산→호치민 해운 15일, 비용... [최종 통합 판단] 세금·인력 측면은 유리하나, 물류 리드타임 증가가 변수...
방법 2: 전문가 검증 방식
"가정으로 명시" 요청 → AI의 할루시네이션(사실처럼 보이는 오류 정보)을 줄임
패턴적합한 상황
명시적 분해 요청질문이 여러 도메인에 걸쳐 있고 변수가 많을 때
전문가 검증 방식특정 전문 지식 기반의 단계적 논증이 필요할 때
하위 질문 수 지정AI의 분해 깊이를 통제하고 싶을 때 (3~5개 등 명시)

메타언어 생성 · 무한 생성
반복 작업을 줄이고 대량 생성하는 기법. 한 번 설계하면 무한히 재사용한다. 단축 명령어로 긴 조건을 대체하고, 변수 조합으로 동일 형식의 결과물을 대량 생성한다.
15. 메타언어 생성 패턴 : AI와 대화하는 나만의 비밀 코드
AI에게 특정 명령어(/명령어)에 대한 행동 지침을 미리 학습시켜, 긴 설명 없이도 원하는 결과물을 즉시 얻어내는 고도의 프롬프트 기법.
핵심 흐름 : [명령어 사전 학습] → [단축 명령어 입력] → [정밀 결과물 즉시 출력]
패턴 A : 기본 명령어 정의 단일 명령어 세트
자주 쓰는 출력 형식을 명령어 하나로 고정. 대화 시작 시 한 번에 학습시키면 해당 대화 내내 유효하다.
아래 명령어를 학습하세요. 이후 해당 명령어 입력 시 정의된 방식으로만 출력하세요. /요약 = 핵심만 3줄, 전문용어 제거, 무역 실무자 기준, [결론 → 이유] 순서로 작성 /코드 = Python 작성, 주석 한국어, 예외처리 포함, 실행 가능한 완성 코드만 출력 (설명 생략) /비교 = 2~4개 대안을 표로 정리. 항목: [장점 | 단점 | 비용 | 난이도 | 추천상황] 마지막 행에 종합 추천 추가 /아이디어 = 창의적이고 실현 가능한 대안 5가지 제시, 각 아이디어별 예상 효과 및 리스크 포함 /검토 = 작성된 내용의 논리적 오류·누락된 점을 비판적으로 분석하고 개선 방향 제시 /가이드 = 초보자가 따라 할 수 있는 Step-by-Step 절차 구성, 단계별 주의사항 포함
AI 예상 응답 (/비교 사용 시)
사용자 입력 : /비교 엑셀 수작업 vs 파이썬 자동화 vs 노코드 툴 | 항목 | 엑셀 수작업 | 파이썬 자동화 | 노코드 툴 | |----------|------------------|-----------------|------------------| | 장점 | 즉시 시작 가능 | 완전 자동화 | 코딩 불필요 | | 단점 | 오류·반복 작업 | 학습 시간 필요 | 기능 제한 | | 비용 | 무료 | 무료 | 월 5~20만원 | | 난이도 | 낮음 | 높음 | 중간 | | 추천상황 | 소량·비정기 | 대량·정기 반복 | 비개발자·중간규모 | | 종합추천 | - | - | ✅ 노코드 툴 우선 추천 |
패턴 B : 변수 기반 만능 명령어 가변 결과물 생성
[대상]과 [옵션]을 바꿔가며 동일 형식의 결과물을 반복 생성. 구조 : /명령어:[대상]:[옵션]
[시스템 설정] [독자] = 초등학생 [역할] = 출판사 편집자 (교양 도서 교정 중) [명령어 학습] /유:[단어]:[개수] = [단어]에 대해 [독자]가 이해할 수 있는 유의어 [개수] 제공 + 가장 추천하는 유의어 선정 /반:[단어]:[개수] = [단어]에 대해 [독자]가 이해할 수 있는 반의어 [개수] 제공 + 가장 추천하는 반의어 선정 /유반:[단어]:[개수] = 유의어와 반의어 검색을 동시에 수행
AI 예상 응답 (/유반:신중하다:2 입력 시)
유의어 (초등학생 수준) 1. 조심스럽다 2. 꼼꼼하다 추천 : '조심스럽다' (일상에서 가장 많이 쓰여요!) 반의어 (초등학생 수준) 1. 덜렁대다 2. 성급하다 추천 : '덜렁대다' (반대 상황을 설명하기 가장 쉬워요!)
패턴 C : 복합 명령 & 운영 팁 Multi-Command
명령어를 조합하면 한 번에 복합 결과물 생성 가능. /리셋으로 대화가 길어졌을 때 맥락을 초기화할 수 있다.
/요약 /비교 FTA와 MFN 관세율의 차이 → 3줄 요약 + 비교표를 동시에 생성 /검토 /가이드 아래 수출 절차 문서를 검토하고 개선된 가이드로 재작성해줘 → 논리 오류 분석 후 Step-by-Step 절차로 재구성 /리셋 → 대화가 길어져 맥락이 꼬일 때 사용. 메타언어 규칙만 남기고 초기화
19. 무한 생성 패턴 : 무한한 맞춤형 출력 생성하기
고정 프레임(변하지 않는 구조)과 변수(바꿀 수 있는 값)를 분리하여 변수 조합만 바꾸면 동일한 형식의 결과물을 무한히 생성하는 패턴. 이메일 템플릿, 상품 설명, 소셜 포스트 등에 특히 강력하다.
핵심 공식 : 변수 조합 수 = 각 변수의 선택지 수를 곱한 값
예) 수신자 4 × 목적 4 × 톤 3 = 최대 48가지 조합 자동 생성 가능
만능 템플릿 구조 고정 프레임 + 변수
기본 구조

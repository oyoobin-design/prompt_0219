13. 질문 개선 패턴 : 질문을 질문하기
AI에게 사용자의 질문 자체를 먼저 분석하고 더 나은 질문으로 업그레이드하게 하는 패턴. 모호하거나 전제가 잘못된 질문을 먼저 교정함으로써 더 정확하고 유용한 답변을 얻는다.
핵심 원리 : 질문의 품질이 답변의 품질을 결정한다. AI에게 "먼저 질문을 분석하라"고 지시하면 맥락 오류를 사전에 차단할 수 있다.
방법 1 : 단순 개선형 개선 후 즉시 답변
아래 질문을 더 명확하고 답변 가능한 형태로 개선한 뒤, 개선된 질문으로 답변해. 개선한 이유도 한 줄로 설명해. 질문 : AI를 무역에 활용하려면 어떻게 해야 하나요?
AI 예상 응답
[개선된 질문] "수출입 서류 작성과 바이어 이메일 업무를 담당하는 중소기업 무역 실무자가 ChatGPT 또는 Claude를 활용하여 반복 업무를 줄이고 싶을 때, 즉시 적용 가능한 활용법 3가지를 단계별로 알려주세요." [개선 이유] '무역'과 'AI 활용'의 범위가 너무 넓어 구체적인 실무 업무와 수준을 명시해야 실행 가능한 답변이 가능하기 때문. [답변] 1. 바이어 이메일 초안 자동 생성 : 거래 조건·제품 정보를 입력하면 영문 이메일 즉시 생성 2. 수출 서류 초안 작성 : 인보이스·패킹리스트 항목을 프롬프트로 구조화하여 초안 생성 3. 관세·인증 정보 요약 : 복잡한 규정 문서를 붙여넣고 핵심만 요약 요청
방법 2 : 다방향 분기형 3가지 해석 → 선택 후 답변
제 질문을 3가지 다른 해석으로 개선해 주세요. 각 버전이 어떤 상황에 맞는지 설명하고, 제가 선택하면 그것으로 답하세요. 질문 : 수출 자동화는 어떻게 시작하나요?
AI 예상 응답
버전 A — 업무 프로세스 자동화 관점 "개발 인력 없이 엑셀·노코드 툴로 수출 서류·이메일 업무를 자동화하는 첫 단계는?" → 적합한 상황 : IT 예산이 적고 빠르게 시작하고 싶은 중소기업 실무자 버전 B — Python 코딩 자동화 관점 "Python 기초를 배운 무역 실무자가 수출 데이터 정리·보고서 생성을 자동화하기 위한 첫 번째 실습 과제는?" → 적합한 상황 : 코딩을 배워서 직접 구현하고 싶은 실무자 버전 C — 물류·통관 자동화 관점 "중소 수출기업이 통관·물류 추적 업무를 자동화하기 위해 활용 가능한 기존 SaaS 툴과 도입 순서는?" → 적합한 상황 : 물류·통관 프로세스 효율화가 우선인 기업 어떤 버전으로 답변드릴까요?
방법 3 : 전제 교정형 잘못된 전제 지적 → 교정된 질문으로 답변
제 질문에 잘못된 전제나 오해가 있으면, 먼저 그것을 지적하고 교정된 질문으로 다시 물어봐 주세요. 그다음 교정된 질문에 답하세요. 질문 : FTA가 적용되면 관세를 안 내도 되는 건가요?
AI 예상 응답
[전제 교정] 이 질문에는 두 가지 잘못된 전제가 있습니다. ① "FTA 적용 = 관세 면제"가 아닙니다. 협정마다 세율이 다르며 단계적으로 인하되는 품목도 많습니다. ② FTA 혜택을 받으려면 원산지 증명서 발급 등 별도 절차가 필요합니다. [교정된 질문] "한-미 FTA 적용 시, 특정 품목의 관세율이 얼마나 인하되며, 혜택을 받기 위해 수출자가 준비해야 할 원산지 증명 절차는 무엇인가요?" [답변] 한-미 FTA 기준, 대부분의 섬유류는 협정 발효 즉시 0% 또는 단계적 철폐...
방법언제 사용
방법 1 (단순 개선)빠르게 개선된 답변이 필요할 때
방법 2 (다방향 분기)내 질문의 방향을 아직 확신하지 못할 때
방법 3 (전제 교정)상식처럼 알고 있지만 확신이 없는 정보를 물어볼 때
14. 팩트체크 목록 패턴 : 정보 검증을 위한 목록 작성하기
AI 답변 안에 포함된 검증 필요 항목을 스스로 추출·목록화하게 지시하는 패턴. 법규·세율·인증처럼 자주 바뀌거나 틀리면 손해가 큰 정보에 필수적으로 활용한다.
핵심 원리 : AI는 확신 있게 틀린 정보도 자신 있게 말할 수 있다(환각 현상). 팩트체크 목록을 요청하면 AI 스스로 불확실한 항목을 표시하게 된다.
방법 1 : 기본 검증 목록형 신호등 색상 분류

AI 예상 응답

16. 리플렉션 패턴 : AI를 위한 자기성찰의 거울
AI가 초안을 작성한 뒤, 스스로 결과물을 검토(비평)하고 수정(개선)하게 함으로써 최종 답변의 품질을 비약적으로 높이는 기법. 환각(Hallucination) 현상을 줄이고 논리적 완결성을 높이는 데 탁월하다.

유형 1: 단순 나열형
아래 문제를 해결하는 방법을 최소 4가지 제시하세요. ## 방법 {번호} : {이름} - 핵심 원리 : {한 줄로 작동 방식 설명} - 적합한 상황 : {언제 이 방법이 가장 효과적인가} - 단점 : {주의해야 할 제약이나 위험 요소} - 난이도 : {초급 / 중급 / 고급} # 문제 수출통관 서류를 자동으로 생성하고 싶습니다.
유형 2: 축 기반 비교형
아래 4가지 방법을 다음 평가 기준으로 비교 분석하세요. 결과는 표 형태로 출력하고 최적 선택지에 ★ 표시를 추가하세요. 평가 기준: 개발 속도 / 유지보수 난이도 / 비용 / 확장성 비교 대상: FastAPI, Flask, Node.js Express, Django 용도: 무역 데이터 처리 REST API 서버
유형 3: 상황별 분기형
아래 각 상황에 맞는 가장 적합한 온라인 마케팅 채널을 추천하세요. 각 상황은 서로 독립적으로 판단하세요. 상황 A : 뷰티 스타트업, 월 광고비 30만원, 20대 여성 타겟, 빠른 인지도 필요 상황 B : B2B SaaS 서비스, 중소기업 의사결정자 대상, 리드 발굴 목적 상황 C : 지역 음식점, 반경 3km 고객 유입, 즉각적인 예약 증가 필요 상황 D : 해외 진출 초기 단계, 영어권 시장, 브랜드 인지도 구축
AI 예상 응답
상황 A 1순위 추천 : 인스타그램 릴스 - 20대 여성 타겟과 정확히 일치, 소액으로도 바이럴 가능 2순위 추천 : 틱톡 광고 - 뷰티 카테고리 알고리즘 친화적, 팔로워 없이도 노출 가능 상황 B 1순위 추천 : 링크드인 광고 - 직책·업종 기반 정밀 타겟팅, 의사결정자 직접 도달 가능 2순위 추천 : 구글 검색광고 - 솔루션 검색 의도를 가진 유저 포착, 전환율 높음 상황 C 1순위 추천 : 네이버 플레이스 광고 - 위치 기반 검색 유입, 예약 시스템 직접 연동 가능 2순위 추천 : 인스타그램 위치 광고 - 반경 내 노출로 방문 의도 높은 유저 타겟 상황 D 1순위 추천 : 구글 디스플레이 광고 - 영어권 글로벌 도달, 브랜딩 캠페인에 최적 2순위 추천 : 링크드인 콘텐츠 마케팅 - B2B 글로벌 신뢰도 구축, 의사결정자 대상 유효
유형 4: 역발상 대안형
이용자 페르소나 패턴
페르소나 패턴이 "AI의 역할"을 정의한다면, 이용자 페르소나는 "이 답변을 듣는 사람이 누구인가"를 정의한다. AI가 청중의 배경·수준·니즈에 맞게 어휘·깊이·예시를 자동으로 조정한다.
이용자같은 질문에 대한 응답 스타일
무역 초보 대학생쉬운 비유, 기초 개념부터, 전문용어 최소화
수출 담당 팀장실무 사례, 비용·리스크 중심, 즉시 활용 가능한 정보
금융기관 IT 결정자기술 아키텍처, 도입 비용·기간, 보안·확장성 중심

이용자 페르소나 설계 체크리스트
프롬프트 작성 전 아래 항목을 점검하면 AI가 더 정밀하게 맞춤형 답변을 생성한다
이 사람의 직업과 역할은?
→ AI가 응답의 전문성 수준과 맥락을 설정하는 기준
이 주제에 대한 사전 지식은?
→ 초보 = 비유 중심 / 전문가 = 기술 용어 직접 사용
이 정보로 무엇을 결정하거나 실행할 예정인가?
→ 이론 중심 vs 실행 중심 중 어느 쪽으로 답할지 결정
지금 어떤 상태인가? (급함 / 여유)
→ 급함 = 결론 먼저, 핵심만 / 여유 = 배경 설명 포함
얼마나 긴 답변을 읽을 여유가 있는가?
→ 여유 없음 = 3줄 이내 / 여유 있음 = 상세 설명 허용
가장 두려워하는 것은? / 가장 원하는 것은?
→ 두려움 = AI가 리스크를 먼저 짚어줌 / 원하는 것 = 목표 향한 답변 구성
절대 쓰면 안 되는 언어나 표현이 있나?
→ 네거티브 프롬프팅으로 명시하면 더 정확하게 적용됨
